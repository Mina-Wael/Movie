# Movie App
It's an android application that shows top rated and upcoming movies and allows user to save favorite movies.

# Tools

* Kotlin
* Coroutine
* MVVM
* Retrofit
* Room database
* Repository Pattern
* Dependency injection (Hilt)
* LiveData
* Flow
* Pagination (paging 3)
* Navigation Component
* Clean architecture
